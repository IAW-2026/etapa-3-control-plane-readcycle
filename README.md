# Control Plane - ReadCycle

Aplicación **Control Plane** del Proyecto IAW 2026 — **ReadCycle**.

Esta aplicación corresponde al panel central de administración del marketplace y permite supervisar, gestionar y coordinar todas las aplicaciones que conforman el ecosistema.

## Aplicaciones integradas

El marketplace está compuesto por las siguientes aplicaciones:


* **[Buyer App](https://buyer-readcycle.vercel.app/)**
  Plataforma utilizada por los compradores para explorar el catálogo, realizar compras y consultar sus pedidos.

* **[Seller App](https://proyecto-c-seller-readcycle.vercel.app/)**
  Plataforma destinada a vendedores para publicar libros, administrar su catálogo y gestionar órdenes.

* **[Payments App](https://proyecto-c-payments-readcycle-nlqt.vercel.app/)**
  Servicio encargado del procesamiento de pagos, validación de transacciones y administración financiera.

* **[Shipping App](https://proyecto-c-shipping-readcycle.vercel.app/)**
  Plataforma responsable de la gestión logística, seguimiento y administración de los envíos.

## Descripción

El **Control Plane** centraliza la administración del marketplace ReadCycle, proporcionando una única interfaz para supervisar el funcionamiento de todas las aplicaciones que forman parte del sistema.

La plataforma permite:

* Supervisar el estado general del marketplace.
* Visualizar métricas globales de todas las aplicaciones.
* Administrar usuarios del ecosistema.
* Gestionar permisos y roles administrativos.
* Acceder rápidamente a cada una de las aplicaciones integradas.
* Monitorear el funcionamiento general del sistema.
* Centralizar la operación administrativa del proyecto.

## Rol del sistema

### Administrador

El Control Plane únicamente puede ser utilizado por usuarios con permisos administrativos.

Puede:

* Gestionar usuarios del sistema.
* Consultar métricas globales del marketplace.
* Supervisar el funcionamiento de todas las aplicaciones.
* Acceder a Buyer, Seller, Payments y Shipping desde una única interfaz.
* Administrar permisos y estados de usuarios.
* Centralizar la operación del ecosistema ReadCycle.

## Funcionalidades principales

* Dashboard administrativo unificado.
* Gestión centralizada de usuarios.
* Métricas generales del marketplace.
* Navegación integrada entre aplicaciones.
* Administración de permisos.
* Supervisión del estado general del sistema.

## Instrucciones adicionales

* Solo los usuarios administradores pueden acceder al Control Plane.
* Para utilizar la plataforma el usuario debe encontrarse activo dentro del sistema.
* El panel funciona como punto central de administración y coordinación del marketplace, mientras que la operación específica de compradores, vendedores, pagos y envíos se realiza dentro de sus respectivas aplicaciones.
